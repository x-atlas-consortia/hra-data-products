## HRA KG

Go to <https://lod.humanatlas.io> to browse the current HRA KG. 

Additionally, [hra-kg.blazegraph.jnl](./hra-kg.blazegraph.jnl) is a Blazegraph journal file generated by the 
[HRA Data Products generator](https://github.com/x-atlas-consortia/hra-data-products) and consists of all 
the named graphs used in queries for reports below. It can be used by the Blazegraph KG software.
